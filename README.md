# markdown을 vscode에서 사용하기

## 목적 
- vscode를 확장 프로그램을 설치하면 문서를 작성하기 편하고, 관리하기 편하다.
- 사용하다 보면 불편한 점도 있지만 markdown 문서에 대한 불편함인 것 같고 vscode에서 사용하는것에 대한 불편이라고 보기는 어렵다.
